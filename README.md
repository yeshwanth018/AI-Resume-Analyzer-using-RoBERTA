# AI-Resume-Analyzer-using-RoBERTA
An AI-powered resume analyzer that matches candidate resumes with job descriptions using the RoBERTa transformer model. The system performs semantic similarity analysis to calculate match scores, identify skill gaps, and improve candidate-job alignment beyond traditional keyword matching.

Tech Stack: Python, RoBERTa, Hugging Face Transformers, NLP, Scikit-learn, Streamlit/Flask
